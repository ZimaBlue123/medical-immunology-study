# 医学免疫学学习系统 - 完整使用指南

## 📚 系统概述

本系统是一个**完整的医学免疫学学习平台**，基于《医学免疫学》人卫第9版，提供：
- 📖 **知识卡片库**：涵盖教材所有核心知识点
- 💬 **智能学习教练**：引导式学习，Socratic方法
- ✏️ **智能练习系统**：自适应出题、错题本、间隔复习
- 📊 **学习进度追踪**：模块掌握度、统计报告
- 🔍 **知识搜索**：快速查找任何知识点

---

## 🚀 快速开始

### Web应用（推荐）

```powershell
# 启动Web应用
python app.py

# 浏览器访问
http://127.0.0.1:5000
```

### 命令行工具

```powershell
# 随机练习10题
python -m immuno_study quiz --deck decks/people9-core.json --n 10

# 复习到期题目
python -m immuno_study review --deck decks/people9-core.json --n 10

# 查看统计
python -m immuno_study stats
```

## 🎯 核心功能详解

### 1. 📚 知识卡片库

**功能特点**：
- ✅ 涵盖12个学习模块，150+核心概念
- ✅ 每个概念包含：定义、要点、记忆技巧、临床关联
- ✅ 支持搜索、按模块浏览、查看详情

**使用方法**：

1. **搜索知识点**
   - 在搜索框输入关键词（如：补体、T细胞、抗体）
   - 点击搜索或按Enter
   - 查看匹配的知识卡片列表

2. **浏览所有知识点**
   - 点击"浏览所有知识点"按钮
   - 可选择特定模块筛选
   - 点击卡片查看详细信息

3. **按模块学习**
   - 点击模块卡片进入模块详情
   - 查看该模块的所有核心概念
   - 查看常见混淆点和临床关联
   - 直接做该模块的练习题

**知识卡片内容**：
- 📖 **核心概念**：术语、定义、关键要点
- 💡 **记忆技巧**：帮助记忆的提示
- ⚠️ **常见混淆点**：易混淆概念的对比
- 🏥 **临床关联**：与临床疾病的联系
- 🎯 **考试重点**：高频考点提示

---

### 2. 💬 智能学习教练

**功能特点**：
- ✅ Socratic方法：先探测基础，再针对性讲解
- ✅ 自适应跟进：根据理解程度调整教学
- ✅ 理解核验：每次讲解后提供检验题

**使用方法**：

1. **开始学习**
   - 输入想学习的主题（如：补体系统、T细胞活化）
   - 或点击快捷按钮选择常见主题

2. **探测阶段**
   - 教练会先问你对主题的理解
   - 你可以描述你的理解，或点击"跳过，直接讲解"

3. **讲解阶段**
   - 教练提供结构化讲解
   - 包含概念定义、要点、记忆技巧
   - 提供常见混淆点和临床关联

4. **巩固阶段**
   - 教练提供理解检验问题
   - 可以点击"做几道题巩固一下"
   - 系统自动跳转到相关练习题

---

### 3. ✏️ 智能练习系统

**功能特点**：
- ✅ 随机抽题：按模块或全部随机
- ✅ 即时反馈：答对/答错立即显示解析
- ✅ 错题本：自动记录错题
- ✅ 间隔复习：基于SRS算法安排复习

**使用方法**：

1. **开始练习**
   - 选择题量（1-25题）
   - 选择模块（可选，默认全部）
   - 点击"开始练习"

2. **答题**
   - 选择题：点击选项
   - 简答题：输入答案后提交
   - 查看解析和学习引导

3. **错题本**
   - 自动记录所有错题
   - 可专门复习错题
   - 答对后自动从错题本移除

4. **复习**
   - 系统根据间隔复习算法安排复习
   - 到期题目会自动提醒
   - 可设置只复习错题

---

### 4. 📊 学习进度追踪

**功能特点**：
- ✅ 模块掌握度：每个模块的练习进度
- ✅ 统计报告：近7天做题情况
- ✅ 错题统计：错题数量和分布

**查看方法**：
- 点击"进度"标签
- 查看各模块的练习进度和掌握度
- 查看近7天的做题统计

## 📖 知识库内容

### 12个学习模块

1. **免疫学总论与基本概念**
   - 固有免疫 vs 适应性免疫
   - 免疫三大功能
   - 免疫系统组成
   - 抗原分类（完全抗原、半抗原、超抗原、佐剂）

2. **固有免疫：屏障、细胞与分子**
   - PAMP/DAMP和PRR
   - TLR、NK细胞、巨噬细胞、DC
   - 细胞黏附分子、趋化因子
   - 炎症反应过程

3. **补体系统**
   - 三条激活途径（经典、旁路、凝集素）
   - 补体片段功能（C3b、C5a、MAC）
   - 补体调节机制
   - 补体受体

4. **MHC与抗原呈递**
   - MHC I和MHC II的区别
   - 抗原呈递的五个步骤
   - 抗原表位分类
   - 交叉呈递

5. **抗体/免疫球蛋白**
   - 抗体结构（Fab、Fc）
   - 五类Ig的特点（IgG、IgM、IgA、IgE、IgD）
   - 特异性与效应功能

6. **T细胞免疫**
   - T细胞活化双信号/三信号模型
   - CD4+ T细胞亚群（Th1、Th2、Th17、Treg）
   - CD8+ T细胞（CTL）
   - 免疫检查点

7. **B细胞免疫与体液免疫**
   - B细胞活化（TD/TI抗原）
   - 生发中心反应
   - 亲和力成熟 vs 类别转换
   - 初次/再次应答

8. **细胞因子**
   - 主要细胞因子（IL-1、IL-2、IL-4、IL-6、IL-10、IL-12、IL-15、IL-17、IL-18）
   - 干扰素（I型、II型）
   - TNF-α、TGF-β
   - GM-CSF

9. **超敏反应**
   - I型（速发型/过敏反应）
   - II型（细胞毒型）
   - III型（免疫复合物型）
   - IV型（迟发型）

10. **免疫耐受与自身免疫**
    - 中枢耐受 vs 外周耐受
    - Treg、AIRE、Foxp3
    - 常见自身免疫病（SLE、RA、1型糖尿病、MS、MG、桥本甲状腺炎）
    - 免疫调节机制

11. **免疫缺陷病**
    - 原发性免疫缺陷（XLA、DiGeorge、SCID、WAS、AT、CGD、LAD等）
    - 继发性免疫缺陷（HIV/AIDS）
    - 各类缺陷与易感病原体

12. **肿瘤免疫与移植免疫**
    - 肿瘤免疫逃逸机制
    - 免疫检查点抑制剂（PD-1、CTLA-4）
    - CAR-T细胞治疗
    - 移植排斥反应（超急性、急性、慢性）
    - 主动/被动免疫、疫苗

## 💡 学习建议

### 推荐学习流程

1. **预习阶段**（10分钟）
   - 浏览模块列表，了解整体框架
   - 选择要学习的模块

2. **学习阶段**（20-40分钟）
   - 使用学习教练，探索主题
   - 查看知识卡片，理解核心概念
   - 注意常见混淆点

3. **练习阶段**（10-20分钟）
   - 做该模块的练习题
   - 记录错题，查看解析

4. **复习阶段**（5-10分钟）
   - 查看错题本
   - 复习到期题目
   - 更新学习进度

1. **利用记忆技巧**
   - 每个概念都有记忆提示
   - 帮助快速记忆关键点

2. **注意混淆点**
   - 系统会标注常见混淆概念
   - 重点对比理解

3. **联系临床**
   - 关注临床关联部分
   - 理解免疫学原理在疾病中的应用

4. **定期复习**
   - 使用间隔复习功能
   - 定期回顾错题

## 📝 常见问题

### Q: 如何添加新的知识点？

A: 编辑 `immuno_study/knowledge.py`，在对应模块的 `key_concepts` 中添加新概念。

### Q: 如何添加新的题目？

A: 编辑 `decks/people9-core.json`，按照现有格式添加新题目。

### Q: 数据存储在哪里？

A: 学习记录存储在 `immuno_study/data/` 目录下：
- `srs.json`：间隔复习状态
- `wrong_cards.json`：错题本
- `attempts.jsonl`：做题记录

### Q: 如何重置学习进度？

A: 删除 `immuno_study/data/` 目录下的文件即可。
