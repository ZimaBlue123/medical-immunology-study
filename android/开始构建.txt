医学免疫学学习系统 - Android应用
========================================

📁 所有Android文件都在此文件夹中！

🚀 快速开始（三种方法任选一种）：

方法1：一键构建（最简单）⭐
  双击运行：一键构建APK.bat
  脚本会自动检测环境并构建APK

方法2：Android Studio（推荐）
  1. 安装Android Studio
  2. File -> Open -> 选择此android文件夹
  3. Build -> Build APK(s)
  4. APK在：app/build/outputs/apk/debug/app-debug.apk

方法3：命令行
  cd android
  gradlew.bat assembleDebug

========================================
APK文件位置（构建后）
========================================

Debug版本：
android/app/build/outputs/apk/debug/app-debug.apk

Release版本：
android/app/build/outputs/apk/release/app-release.apk
